1. Informazioni generali

QtDrive è un’applicazione che consente agli utenti di gestire i loro account di servizi cloud ed i relativi file, consentendo la visualizzazione e la modifica di informazioni utili.

L’applicazione supporta il salvataggio ed il caricamento di dati tramite opportuni file in formato XML.

 

 

2. Funzionalità del menù

Il menù collocato nella parte superiore della finestra è suddiviso in due categorie: Profilo e Aiuto.

2.1 Profilo

Nuovo

Consente di creare un nuovo file all'interno del quale memorizza le informazioni dei file.

Apri

Consente di caricare le informazioni degli account gestiti da QtDrive da un opportuno file in formato XML. All'avvio dell'applicazione, di default, viene creato il file account.xml.

Salva

Salva eventuali modifiche effettuate sul file correntemente aperto.

Salva con nome

Consente di effettuare il salvataggio delle informazioni degli account all’interno di un nuovo file.

Esci

Chiude l’applicazione.

 

2.2 Aiuto

Guida

Consente di caricare le informazioni degli account gestiti da QtDrive da un opportuno file in formato XML. All'avvio dell'applicazione, di default, viene creato il file account.xml.

Informazioni su...

Visualizza informazioni generali sull’applicazione.

 

 

3. Interfaccia principale

La finestra principale dell’applicazione è costituita da tre schede, Account, File e Ricerca, selezionabili tramite un singolo clic del mouse.

3.1 Account

La scheda Account consente all’utente di inserire account di servizi cloud da gestire con l’applicazione e di visualizzare gli altri già presenti.

3.1.1 Inserimento

Per inserire un nuovo account è sufficiente premere il pulsante Aggiungi account, collocato in basso a sinistra. L’utente deve specificare l’indirizzo email, la password, lo spazio a disposizione e selezionare l’host del servizio dal menù a tendina.

 

3.1.2 Visualizzazione

Nella scheda account vengono visualizzati tutti gli account gestiti dall’applicazione, tramite un singolo clic su uno di essi è possibile aprire una schermata per la visualizzazione di informazioni aggiuntive, come il numero di file contenuti e lo spazio rimasto a disposizione.

Premendo il pulsante Elimina è possibile rimuovere l‘account, tuttavia il suo contenuto non sarà più gestibile tramite l‘applicazione.

3.1.3 Modifica

Selezionando invece il pulsante Modifica è possibile modificare l‘indirizzo email e la password associati all‘account selezionato.

3.2 File

Da questa scheda è possibile visualizzare i file contenuti all’interno degli account, inserirne di nuovi, modificarli ed eliminarli. Per visualizzare i file di un account è sufficiente selezionare quest’ultimo dalla lista presente sullo schermo.

3.2.1 Inserimento

Per inserire un nuovo file è necessario premere il pulsante Nuovo file e compilare il form che compare. Le informazioni da inserire varieranno a seconda del tipo di file che si vuole caricare.

3.2.2 Visualizzazione

Selezionando dalla lista un file con un doppio clic, si apre una finestra che consente all’utente di visionare ulteriori informazioni come le date di creazione e di caricamento o altri dati dipendenti dal tipo del file.

3.2.3 Modifica

Dalla finestra di informazioni del file è possibile anche modificare il nome del file e la sua descrizione. Effettuate le modifiche, per salvare occorre premere il pulsante Conferma.

 

3.2.4 Eliminazione

Per eliminare un file è necessario selezionarlo con un singolo clic dalla lista di file visualizzata a schermo. Il pulsante Elimina può a questo punto essere premuto: a seguito di un’ulteriore conferma, il file verrà rimosso.

 

3.3 Funzionalità di Ricerca

L’applicazione fornisce all’utente utili funzioni che consentono di ricercare file all’interno di tutti gli account che egli ha registrato.

La ricerca può avvenire per nome del file, per descrizione o per informazioni aggiuntive (come bitrate nel caso di file audio, risoluzione nel caso di immagini, ...) e può essere anche case-sensitive. Le impostazioni di ricerca vengono specificate dall’utente. Nel caso in cui nessuna opzione sia stata specificata, si attiverà in automatico la ricerca per nome.

Una volta trovato il file di interesse, tramite un doppio clic, si possono visualizzare le informazioni del file.
